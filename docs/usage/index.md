# Usage Guide

Learn how to set up and use matrixd for agent-to-Matrix integration.

## Workflow

1. **Install** matrixd (`pip install matrixd`)
2. **Configure** your homeserver, token, and room policies
3. **Verify** credentials with `matrixd whoami`
4. **Run** the listener or use CLI commands
