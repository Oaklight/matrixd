# 使用指南

了解如何设置和使用 matrixd 将 AI Agent 接入 Matrix。

## 工作流程

1. **安装** matrixd（`pip install matrixd`）
2. **配置** homeserver、token 和房间策略
3. **验证** 凭据：`matrixd whoami`
4. **运行** 监听器或使用 CLI 命令

## 章节

- [安装](installation.md) — 从 PyPI 或源码安装
- [配置](configuration.md) — JSONC 配置文件参考
- [监听器](listener.md) — 运行带策略过滤的事件监听器
